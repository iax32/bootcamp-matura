CSS3

CSS3 bestimmt das Aussehen einer Webseite. Wird meist in einer einziegen Datei zusammengefasst. kann sowohl normale HTML tags/Elemente wie <div> ansprechen, als auch Klassen wie .class oder IDs wie #ID. Klassen werden für mehrere Elemente verwendet, während IDs einziegartig sein sollten.

Box-Model

In HTML/CSS hat jeder Inhalt selbst eine Breite und Höhe. Dann werden zwiebelartig immer weitere Boxen rundherum gelegt. Dies findet in der folgenden Reihenfolge statt: padding -> border -> margin
Diese können in bottom, top, left und right aufgeteilt werden.

Fluid Grid

Ein fluid grid sorgt dafür, dass sich eine Webseite an die Bildschirm- bzw. Anwendungsgröße anpasst. Es erreicht dies damit, dass es Spalten mit einer relativen Breite (bei 12 Spalten 8.3%) erschafft ( im Vergleich: responsive grid = absolute Breite).

Dies führt zu einer organischen Skalierung, kann jedoch bei kleineren Bildschirmen überwältigend wirken und sollte daher br mit einem responsivem Grid kombiniert werden. Hierführt unbedingt notwendig ist die CSS-Einstellung: 

border-sizing: border-box;
Für alle Elemente

B - Dynamische Webseiten

Statische Webseiten

Auf statische Webseiten wird jedem Benutzer der gleiche Inhalt angezeigt und Veränderungen auf der Webseite können nur durch Veränderungen im Quelcode ausgelöst werden

HTML und CSS

HTML und CSS bilden gemeinsam statische Webseiten. Diese zeichnen sich dadurch aus, dass Sie sehr simpel sind und im Endeffekt nicht als echte Programmiersprache bezeichnet werden können. Viel eher sind sie eine Art Text zu formatieren. CSS kann auch in dynamischer Weise verwendet werden.

Dynmaische Webseiten

Auf Dynmaischen Webseiten hat jeder Benutzer eine andere Sicht. Die Webseite kann zum Beispiel auf Datenbanken zurückgreifen, um jemanden zu identifizieren und so einen anderen Inhalt anzuzeigen. Auch Scripts mit Alterts oder Ähnlichem können vorkommen.

PHP

PHP ist eine Programmiersprache, welche vorallem zur Kommunikation mit Datenbankservern (über SQL) verwendet wird. Die Scripts werden alle am Server ausgeführt und sind für den Benutzer nicht auslesbar. Sie ist ein wichtiger Bestandteil dynamischer Webseiten.

Javascript

Javascript sorgt dafür, dass Webseiten interaktiv und dynamisch gestaltet werden. Scripts in JS werden auch am Client ausgeführt und sind daher auch für den Benutzer auslesbar.

B - Praxis Beispiele

Zur Praxis muss ein Verbindungscript mit Php erstellt werden.

Es muss zuerst XAMPP installiert werden auf dem Gerät. Danach navigiert man zum Ordner C:\xampp\htdocs und fügt dort seine index.php datei hinzu der folgenden Verbindungsscript enthält:


<?php
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "testdb";

$conn = new mysqli($servername, $username, $password, $dbname);

if ($conn->connect_error) {
    die("Verbindung fehlgeschlagen: " . $conn->connect_error);
} else {
    echo "Verbindung erfolgreich!";
}
?>

In xampp wird dann bei den Modulen Apache und Sql auf start gedrückt. Beim Modul SQL wird auf Admin geklickt. Dort wird unter "neu" eine neue Datenbank erstellt. Ersetzte $dbname = "testdb" mit dem namen der erstellten Datenbank. Danach wird im Webbrowser localhost/index.php geöffnet.
