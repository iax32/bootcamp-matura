CSS3

CSS3 bestimmt das Aussehen einer Webseite. Wird meist in einer einziegen Datei zusammengefasst.
kann sowohl normale HTML tags/Elemente wie <div> ansprechen, als auch Klassen wie .class oder IDs wie #ID.
Klassen werden für mehrere Elemente verwendet, während IDs einziegartig sein sollten.

Box-Model

In HTML/CSS hat jeder Inhalt selbst eine Breite und Höhe. Dann werden zwiebelartig immer weitere Boxen rundherum gelegt.
Dies findet in der folgenden Reihenfolge statt: padding -> border -> margin
Diese können in bottom, top, left und right aufgeteilt werden.

Fluid Grid

Ein fluid grid sorgt dafür, dass sich eine Webseite an die Bildschirm- bzw. Anwendungsgröße anpasst.
Es erreicht dies damit, dass es Spalten mit einer relativen Breite (bei 12 Spalten 8.3%) erschafft
( im Vergleich: responsive grid = absolute Breite).

Dies führt zu einer organischen Skalierung, kann jedoch bei kleineren Bildschirmen überwältigend wirken und sollte daher br
mit einem responsivem Grid kombiniert werden. Hierführt unbedingt notwendig ist die CSS-Einstellung: 

border-sizing: border-box;
