A - Einsatz von Bibliotheken

Nutzen von Bibiliotheken

Bibliotheken sind sehr wichtig und hilfreich bei der Programmierung. Sie sind eine Sammlung an Modulen aus anderen Projekten. Je nach Sprache bieten sie unterschiedliche Funktionen an. Es gibt drei unterschiedliche Arten von Bibliotheken:
- Die Standardbibliothek,
- eigene Bibliotheken und
- Bibliotheken von Drittanbietern.
Sie helfen genauso wie Funktionen bei der Modularisierung von Projekten. Code muss durch Bibliotheken nicht jedes Mal neu geschrieben werden, sondern es können einfach die Methoden aus der Bibliothek verwendet werden. Beim Import gibt es viele Möglichkeiten, wie z.B. der Import von nur wenigen Methoden oder der Import eines Moduls mit einem anderen Namen. Es gibt auch Pakete, welche importiert werden können

Beispiele

Als Beispiel für Module aus Bibliotheken dient z.B. das Modul math. Dieses hat viele Operationen aus der Mathematik. So bietet es Funktionen wie math.pi, math.cos oder math.sin an. Ein weiteres Modul ist das Modul os, welches einige Interaktionen mit dem Betriebssystem möglich macht. Das Modul random bietet Funktionen an, mit denen man z.B. zufällige Zahlen finden kann.

B - Fachkonzepte Klassen

Klassen

Klassen sind ähnlich wie Methoden, bzw. Funktionen. Auch sie speichern Code in ihnen, der in anderen Segmenten aufgerufen werden kann. Eine Klasse besteht aus mehreren Funktionen. Sie dient zur besseren Organisation eines Projektes. Eine Klasse ist selbst ein Objekt und kann dementsprechend Attribute haben. Als Beispiel dient hier eine Klasse, die ein Konto repräsentiert


class Person:
  def __init__(self, name, age):
    self.name = name
    self.age = age

p1 = Person("John", 36)

print(p1.name)
print(p1.age)

Sie besteht wie man sehen kann aus einem Konstruktor (__init__), und aus mehreren Funktionen, welche Aktionen mit dem Objekt ausführen. Es ist auch möglich Elemente aus einer Klasse in eine andere Klasse zu vererben. Hierfür muss bei der Definierung der Klasse die Oberklasse in der Klammer angegeben werden. Auch die Vererbung aus mehreren Klassen ist hiermit möglich
