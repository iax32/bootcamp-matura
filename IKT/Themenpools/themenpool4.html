A - Arten von Software

Software

Software kann generell definiert werden als eine Ansammlung von Befehlen die einen Computer
gewisse Aufgaben ausführen lassen. Diese können in Formen von Skripten, Programmen,
Applikationen vorkommen.

Systemsoftware

Als Systemsoftware wird die Software bezeichnet, die die Hardware des Computers kontrolliert und
essentielle Funktionen bietet. Daher kann ein Computer nicht ohne Systemsoftware funktionieren. Der
Benutzer muss jedoch nicht wissen, wie die Software funktioniert, um den Computer zu verwenden.
Beispiele sind hier u.a. OS, Driver oder Sprachprozessoren. Generell sind derartige Softwares in low-
level Programmiersprachen wie Assembly oder C verfasst. Teilweise werden auch mid-level Sprachen
wie C++ verwendet.

Anwendungssoftware

Als Anwendungssoftware hingegen kann Software bezeichnet werden, welche eine gewisse
Aufgabe erfüllen soll. Diese werden vom Benutzer bedient und der Benutzer muss wissen, wie sie
funktionieren. Auch sie sind essentiell, jedoch nicht für den Computer als Ganzes, sondern nur für die
jeweilige Aufgabe, für die sie entwickelt wurden. Sie sind meist in high-level Sprachen wie Python
oder mid-level Sprachen wie C++ geschrieben. Sie können nicht unabhängig funktionieren, sondern
brauchen Systemsoftware.

Supportsoftware

Supportsoftware hingegen ist Software, welche andere Software in irgendeiner Form unterstützt. So
kann sie z.B. ein Patch für eine Anwendungssoftware sein. Im Endeffekt kann Supportsoftware nur
eine andere Software modifizieren. Alleingestellt hat sie jedoch keine Funktion.

Malware

Malware steht kurz für ‚malicious software‘ und kann lt. der Internationalen Konferenz für Wissensbasierte Intelligente Informations- und Ingenieurssysteme in fünf Kategorien eingeteilt werden. Diese sind:

1. Backdoors

Es wird eine Möglichkeit installiert, sich ohne Authentifikation auf das infizierte
Gerät Zugang zu verschaffen.

2. Rootkits

Das Hauptziel eines Rootkits ist es unentdeckt, von sowohl dem Benutzer als auch von Sicherheitsprogrammen, zu bleiben. Meist handelt es sich bei Rootkits um Spyware.

3. Trojaner

Dieser versucht den Benutzer davon zu überzeugen, dass er Cleanware und bringt
ihn so dazu, dass er ihn herunterlädt und die nötigen Berechtigungen für das Ziel der Malware gibt.
Die Endziele eines Trojaners sind viele, es kann sich um Spyware handeln, er kann das Gerät zu einem
Botnet hinzufügen etc.

4. Viren

Ein Virus macht mehrere Kopien von sich selbst auf dem Zielgerät, um so die
Desinfektion des Zieles zu erschweren. Für eine derartige Kopie ist es jedoch notwendig, dass der
Benutzer ein gewisses Programm ausführt.

5. Würmer

Dieser ist vor allem daher so gefährlich, weil er sich, durch das Ausnutzen von
Schwachstellen in einem Netzwerk, rapide in jenem verbreitet. Er repliziert sich auf anderen Geräten
im Netzwerk, verglichen mit dem Virus jedoch macht er dies vollautonom.

B - Der Handel mit Daten

Datenhandel

Viele große Tech Firmen verdienen sehr viel Geld mit dem Handel von Daten. Die Daten werden
unter anderem bei der Verwendung ihrer Websites , oder teilweise mithilfe von sogenannten Trackern
geschürft und dann an Firmen wie z.B. Versicherungsfirmen verkauft.

Zweck

Der Handel mit Daten hat viele Vorteile für Firmen:
1. Sie können bei nicht personenbezogenen Daten bessere Marktanalysen machen und so bessere Strategien einsetzen.
2. Bei personenbezogenen Daten können Versicherungsfirmen z.B. die Prämien dementsprechend anpassen, oder Werbefirmen
gezieltere Werbungen an die Zielperson senden.

Preise

Die Preise von Daten sind von vielen Faktoren beeinflusst und variieren daher sehr stark. Für basische
Daten wie Alter, Geschlecht und Ort sind Preise von ca. USD 0.5 für 1000 Personen üblich. Deutlich
wertvoller werden Daten nach gewissen Meilensteilen im Leben. So sind z.B. Daten über Scheidungen
oder Hochzeiten sehr beliebt. Auch Geburten oder ein neues Eigenheim sind wichtige Daten. Für
Daten bezüglich spezieller Krankheiten oder Allergien werden USD 0.26 pro Person verlangt.

Legalität

Der Handel mit vielen Daten ist grundsätzlich legal. So ist z.B. der Handel mit Adressdaten auch ohne die Zustimmung des Betroffenen, durch berechtigtes Interesse der Firmen, legal. Oft verwenden
Unternehmen den Fakt, dass sich die Wenigsten AGBs vollständig durchlesen und verstecken daher
in ihnen gerne Zustimmungen zur Weitergabe von Daten an Dritte. Öffentlich zugängliche Daten
dürfen auch zu Geschäftszwecken gespeichert, genutzt, verarbeitet und übermittelt werden (z.B.
Adresse, E-Mail-Adresse, Telefonnummer aus öffentlichen Verzeichnissen oder Telefonbüchern)

Sensible und personenbezogene Daten

Zu den personenbezogenen Daten zählen alle Daten, welche mit einer bestimmten Person in
Verbindung gebracht werden können. Sensible Daten sind Daten wie die ethnische Herkunft,
politische Meinung, spirituelle Überzeugung, Gesundheit oder das Sexualleben. Unsensible Daten
hingegen sind Daten wie Adresse, Telefonnummer, Beruf oder Geburtsdatum.
