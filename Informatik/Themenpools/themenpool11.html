A - Die Funktionsweise des Internets

Die Funktionsweise des Internets

Das Internet ist ein weltweites Netzwerk von Computern, das es ermöglicht, Informationen und Daten auf der ganzen Welt zu teilen und zu kommunizieren. Es basiert auf der Verwendung des Internetprotokolls (IP) und anderen technischen Standards, um den Austausch von Informationen zwischen verschiedenen Geräten und Computernetzwerken zu ermöglichen. Die Funktionsweise des Internets kann in mehrere grundlegende Schritte unterteilt werden:  Geräteverbindung: Um auf das Internet zugreifen zu können, benötigt man ein Gerät wie einen Computer, ein Smartphone oder ein Tablet, das mit dem Internet verbunden ist. Die Verbindung kann über verschiedene Technologien wie Ethernet, WLAN oder Mobilfunk hergestellt werden.  IP-Adressierung: Jedes Gerät, das mit dem Internet verbunden ist, erhält eine eindeutige IP- Adresse. Diese Adresse ermöglicht die Identifizierung und das Routing von Datenpaketen im Internet. Es gibt zwei Arten von IP-Adressen: IPv4, die aus vier Zahlenblöcken besteht, und IPv6, die aus acht Blöcken von Hexadezimalzahlen besteht. Die IP-Adresse eines Geräts kann entweder statisch sein (fest zugewiesen) oder dynamisch zugewiesen werden (z.B. über DHCP).  Datenpakete und Routing: Informationen im Internet werden in kleine Datenpakete aufgeteilt, um sie effizient über das Netzwerk zu transportieren. Jedes Datenpaket enthält sowohl die Absender- als auch die Empfänger-IP-Adresse sowie den eigentlichen Dateninhalt. Die Datenpakete werden über Router und Switches durch das Internet geleitet. Diese Geräte verwenden Routing-Algorithmen, um die beste Route für die Datenpakete zu finden und sicherzustellen, dass sie ihr Ziel erreichen.  Protokolle: Um die Kommunikation im Internet zu ermöglichen, werden verschiedene Protokolle verwendet. Das Internetprotokoll (IP) ermöglicht das Routing der Datenpakete, während das Transmission Control Protocol (TCP) für die zuverlässige Übertragung der Daten verwendet wird. Das Hypertext Transfer Protocol (HTTP) wird für den Austausch von Webseiten und Inhalten verwendet, während das Simple Mail Transfer Protocol (SMTP) für den Versand von E-Mails zuständig ist. Es gibt viele weitere Protokolle, die spezifische Funktionen im Internet unterstützen. o Das TCP/IP-Referenzmodell, auch als Internet-Protokoll-Suite bezeichnet, ist eine Reihe von Protokollen, die für die Kommunikation in Computernetzwerken verwendet werden. Das Modell besteht aus mehreren Schichten, die jeweils spezifische Funktionen für die Übertragung von Daten über das Netzwerk bieten. Jede Schicht des TCP/IP-Modells hat ihre eigenen Protokolle und Aufgaben. Hier sind die vier Hauptschichten des TCP/IP- Modells: Netzwerkzugangsschicht (Network Access Layer) / Internet-Schicht (Internet Layer) / Transport-Schicht (Transport Layer) / Anwendungsschicht (Application Layer) o Bei einer HTTP-Anfrage sendet dein Webbrowser eine Anfrage an den Server, um Informationen anzufordern oder eine Aktion auszuführen. Die Anfrage enthält eine Methode, eine URL und einen Header. Der Server verarbeitet die Anfrage und sendet eine HTTP-Antwort zurück, die den Statuscode, den Header und gegebenenfalls den Body enthält.  Webseiten und Dienste: Das Internet besteht aus einer Vielzahl von Webseiten, Diensten und Ressourcen, die über das World Wide Web zugänglich sind. Webseiten werden mit Hilfe von Hypertext Markup Language (HTML), Cascading Style Sheets (CSS) und anderen Technologien erstellt. Durch das Eingeben einer URL (Uniform Ressource Locator) in einen Webbrowser können Benutzer auf Webseiten zugreifen und Informationen abrufen.

B - Responsive Webdesign

Responsive Webdesign

Responsive Webdesign ist eine Gestaltungsmethode für Websites, bei der das Layout und die Inhalte flexibel angepasst werden, um sich verschiedenen Bildschirmgrößen und Geräten anzupassen. Das Ziel besteht darin, eine optimale Benutzererfahrung unabhängig davon zu bieten, ob die Website auf einem Desktop-Computer, einem Tablet oder einem Smartphone angezeigt wird. Die Funktionsweise von Responsive Webdesign basiert auf dem Einsatz von flexiblen Rasterlayouts, flexiblen Bildern und Medienabfragen. Hier sind die grundlegenden Konzepte:  Flexibles Raster: Statt fester Breiten für Layoutelemente werden relative Einheiten wie Prozentsätze oder die CSS-Einheit "em" verwendet. Dadurch können sich die Elemente entsprechend der Bildschirmbreite des Geräts anpassen. Das Raster kann in Spalten aufgeteilt werden, die sich je nach verfügbarer Breite neu anordnen.  Flexible Bilder: Bilder werden so skaliert, dass sie sich proportional zur verfügbaren Bildschirmbreite verändern. Dies verhindert, dass Bilder zu groß oder zu klein auf verschiedenen Geräten angezeigt werden. Die Verwendung von CSS-Eigenschaften wie "max-width: 100%" hilft, überlaufende Bilder zu verhindern.  Medienabfragen (Media Queries): Medienabfragen sind CSS-Regeln, die es ermöglichen, unterschiedliche Stile basierend auf bestimmten Eigenschaften des Geräts anzuwenden, wie z.B. Bildschirmbreite, Auflösung und Geräteart. Durch das Hinzufügen von Medienabfragen können spezifische Stiländerungen für verschiedene Gerätegrößen definiert werden. Mit Responsive Webdesign kann eine einzige Website erstellt werden, die auf verschiedenen Geräten optimal funktioniert. Statt separate mobile Versionen oder Apps für verschiedene Plattformen zu erstellen, wird die Website so gestaltet, dass sie sich dynamisch an unterschiedliche Bildschirmgrößen und Auflösungen anpasst. Dies verbessert die Benutzererfahrung, da Benutzer unabhängig vom genutzten Gerät den gleichen Inhalt und die gleiche Funktionalität erhalten. Durch die zunehmende Verbreitung von mobilen Geräten ist Responsive Webdesign zu einer wichtigen Praxis geworden, um sicherzustellen, dass Websites auf allen Geräten optimal angezeigt werden. Es ermöglicht eine bessere Zugänglichkeit, erhöht die Reichweite der Website und trägt zur Verbesserung der Suchmaschinenoptimierung bei.

Das HTML5 Konstrukt

Im Zusammenhang mit Responsive Webdesign bezieht sich das HTML5-Konstrukt auf die Verwendung bestimmter HTML5-Elemente und -Techniken, die dazu beitragen, eine Webseite für verschiedene Bildschirmgrößen und Geräte anzupassen. Hier sind einige wichtige Aspekte des HTML5-Konstrukts im Bezug auf Responsive Webdesign:  Semantische HTML5-Elemente: HTML5 bietet eine Reihe von semantischen Elementen wie <header>, <nav>, <main>, <section>, <article>, <aside>, <footer> usw. Diese Elemente ermöglichen eine klarere Strukturierung des Inhalts und erleichtern die Anpassung des Layouts an unterschiedliche Bildschirmgrößen.  Fluid Grids (flüssige Gitter): Statt fester Breiten für Layout-Elemente werden prozentuale Breiten verwendet, die sich automatisch an den verfügbaren Platz anpassen. Dadurch können sich die Elemente auf unterschiedlichen Bildschirmgrößen flexibel skalieren.  Responsive Formulare: HTML5 bietet neue Formularelemente wie <input type="tel">, <input type="email">, <input type="url">, die eine bessere Handhabung von Benutzereingaben auf verschiedenen Geräten ermöglichen. Zusätzlich kann die Validierung von Formularen mit HTML5- Attributen wie required und pattern verbessert werden.

Das CSS3 Boxmodell

Das CSS3 Boxmodell ist eine Methode, mit der die Darstellung und das Layout von HTML-Elementen gesteuert werden können. Es beschreibt, wie jeder HTML-Elementrahmen (Box) aufgebaut ist und wie sich Eigenschaften wie Rand, Padding und Abstand auf die Größe und Position des Elements auswirken. Das Boxmodell besteht aus vier Hauptkomponenten:  Inhalt (Content): Der eigentliche Inhalt des Elements, wie Text, Bilder oder andere eingebettete Elemente.  Padding: Ein transparenter Bereich um den Inhalt, der zwischen dem Inhalt und dem Rand liegt. Das Padding kann verwendet werden, um den Abstand zwischen dem Inhalt und dem Rand der Box zu kontrollieren.  Rand (Border): Eine Linie, die die Kanten des Elements umgibt und es vom umgebenden Inhalt trennt. Der Rand kann verschiedene Stile, Farben und Dicken haben.  Abstand (Margin): Ein transparenter Bereich um den Rand der Box, der zwischen dem Element und anderen Elementen liegt. Das Margin kann verwendet werden, um den Abstand zwischen den Elementen auf der Seite zu steuern. Diese Komponenten werden in einer hierarchischen Struktur angeordnet, wobei der Inhalt im Zentrum steht und von Padding, Rand und Abstand umgeben wird.Das CSS3 Boxmodell ermöglicht die Kontrolle über die Größe, den Abstand und die Positionierung von Elementen auf einer Webseite. Durch die Anpassung der Werte für Padding, Rand und Abstand kann das Erscheinungsbild der Boxen geändert und das Layout der Webseite gestaltet werden.
